## 📱 PWA支持
- 安装到主屏幕
- 离线访问
- 更快加载速度

## ✨ 主要功能
- 简单的RSS订阅源管理
- 自定义显示选项与主题
- 动态背景与特效
- 响应式设计，支持各种设备

## 🚀 快速开始
1. 访问 [chenli.xin](https://chenli.xin)
2. 添加您喜欢的RSS订阅源
3. 调整显示设置
4. 享受您的个人资讯流

## 🛠 技术栈
- Next.js + React
- TailwindCSS
- PWA
